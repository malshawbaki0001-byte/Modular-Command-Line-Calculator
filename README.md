# Python Modular Command-Line Calculator

Course	EE202: Object-Oriented Programming
Assignment Title	Building a Modular Command-Line Calculator
Due Date	Thursday 16 October 2025


This project is a group assignment for the EE202: Object-Oriented Programming course at King Abdulaziz University (KAU). The goal is to develop a modular, console-based calculator application in Python that emulates the Standard, Programmer, Scientific, and Converter modes of a modern calculator.
